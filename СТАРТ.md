

## Шаг 2 — Настройка .env

Открой файл `.env` в корне проекта и укажи свои данные PostgreSQL:

```
DB_HOST=localhost
DB_PORT=5432
DB_NAME=wishlist
DB_USERNAME=postgres
DB_PASSWORD=4321
```

---

## Шаг 3 — Запуск бэкенда

Открой терминал в корне проекта (там где лежит go.mod) и выполни:

```
go build -o bin/app.exe ./cmd/app/...
.\bin\app.exe
```

Миграции применятся автоматически. Сервер запустится на http://localhost:8081

---

## Шаг 4 — Установка зависимостей фронтенда (только первый раз)

Открой второй терминал и выполни:

```
cd frontend
npm install
```

---

## Шаг 5 — Запуск фронтенда

```
cd frontend
npm start
```

Откроется браузер на http://localhost:3000
